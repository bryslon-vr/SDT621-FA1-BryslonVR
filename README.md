# SDT621-FA1-BryslonVR
This formative assessment demonstrates core C# programming skills through console and Windows Forms applications, focusing on object-oriented design, input validation, and error handling. It includes tasks such as a student results calculator, ATM simulation, programming language manager, municipal service system, and digital identity processor, all designed to solve real-world problems using structured logic and user interaction.

Section A: Short Programming Tasks
Question 1: Student results calculator (average, total, pass/fail)
Question 2: ATM simulation (withdrawal system with balance update)
Question 3: Windows Forms app to manage programming languages
 - Add languages
 - Prevent duplicates and empty input
 - Remove selected items
 - Display date and time

Section B: Long Programming Tasks
Question 1: Emfuleni Municipality Console Application
 - A console-based system that:
 - Captures resident information
 - Records service requests
 - Calculates urgency (priority × severity)
 - Processes requests interactively
 - Generates reports and final summary
Question 2: Digital Identity Processor (Windows Forms)
 - A GUI application that:
 - Captures user details (name, ID, citizenship)
 - Validates South African ID numbers
 - Calculates age from ID
 - Displays validation results
 - Generates a formatted profile summary with timestamp
